# coding_book_shop

This is a coding book webiste built which offers books at a good prices

# Technology used

Frontend: React, TypeScript

Backend: Spring Boot, Java

Database: PostgreSQL

# How to run the project

To run the frontend, go to frontend folder, install dependencies and run the program with command

```
cd frontend
npm install
npm run dev
```

The frontend will be accessible through http://localhost:5173/

To run the backend, locate to the SpringBootLibraryApplication.java file and click run button in your IDE

The backend will be accessible through https://localhost:8080/
